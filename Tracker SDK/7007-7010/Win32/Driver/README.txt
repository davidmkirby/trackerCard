Electro-Optical Imaging, Inc.
Model 7007 PCI, 7010 PCIe and 7410 PCIe Video Tracker
Software Development Kit


This directory contains the Windows UPnP drivers for the
Model 7007/7008, 7010/7011 and 741x Video Trackers.  The drivers
support Windows 2000/XP/Vista/7 both 32-bit and 64-bit versions.

To install the driver, install the video tracker and
boot the computer.  Windows should detect the new hardware
device with a "New Hardware Found" message box.  Acknowledge
this message box.  When Windows displays the "Add New Hardware"
Wizard, point it to the TrkrPci.inf file in this directory.
