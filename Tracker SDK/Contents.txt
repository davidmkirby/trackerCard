Moog, Inc. / Electro-Optical Imaging
Series 7000 and 8000 Video Trackers
Software Development Kit (SDK)


Model 7005 and 7015 VMEbus Tracker
==================================
- Useful C++ classes (targeted towards MS Visual C++)
    * Tracker serial communications
    * Tracker VME communications (GEFanuc/VMIC and Xycom)
    * Tracker TCP/IP communications
    * Brandywine GPS/IRIG VME support (GEFanuc/VMIC and Xycom)
    * Helper classes for tracker messaging
- Message structure header file (useful for both C and C++)
- C and C++ example source code
